# AI粗編(ジェットカット) ✂️

動画/音声ファイルの**無音部分を自動検出**して粗編集(ジェットカット)をつくり、
**Premiere Pro に読み込める FCP XML / EDL** を書き出すツールです。

## 使い方

1. 動画または音声ファイルをドロップ(ファイルは端末内で解析され、**外部には送信されません**)
2. 波形上で検出結果を確認。しきい値・間の余白・最小クリップ長はスライダーで調整
3. セグメントをタップで採用/除外の切り替え、▶で音声プレビュー
4. 「FCP XML」を書き出して Premiere Pro で「ファイル → 読み込み」
   - 読み込み時にメディアのリンク切れが出たら、元ファイルを指定して再リンクしてください
   - EDL / CSV でも書き出せます

## AI選抜(β)

文字起こし(SRT)と指示文を貼り付けると、Claude API が「使うべき範囲」を選抜します。

- 自分の Anthropic API キーが必要です(端末の localStorage にのみ保存)
- 「無音検出と組み合わせる」ONで、無音カット済みセグメントとAI選抜範囲の重なりを採用します
- この機能のみネット接続が必要です。無音検出・書き出しは完全オフラインで動作します

## 技術メモ

- Web Audio API `decodeAudioData` で音声を取り出し、RMS(dB)で無音判定
- 書き出しは FCP7 xmeml v4(クリップの in/out をフレーム単位で指定)、CMX3600風EDL、CSV
- 単一HTML + PWA。ビルド不要、オフライン動作(AI選抜を除く)
