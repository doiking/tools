# 人数カウント 👥

Webカメラの映像から写っている人数をAIでカウントするツール。
イベント会場の集客把握を想定。映像は**端末内で処理され、外部送信・録画保存はしない**。

## 使い方

1. 「📷 カメラ開始」→ 会場にカメラを向ける(初回はカメラ許可が必要)
2. 検出された人に黄色い枠が付き、右上に現在人数が出る(ちらつき防止のため直近7回の中央値)
3. 「⏺ 記録」で一定間隔(1秒〜1分)の自動記録開始 → グラフ・最大・平均を表示
4. 「CSV保存」で `日時,人数` のCSVを書き出し(タッチ端末は共有シート)
5. 「🖼 写真」で会場写真からのカウントも可能

## 精度の注意

- 混雑して人が重なると**実際より少なく**出る(検出上限60人/フレーム)。目安として使うこと
- 「しきい値」を下げると小さく写った人も拾うが誤検出も増える(既定35%)
- 遠すぎる人(画面高の5%以下程度)はほぼ検出できない。カメラは会場を見渡せる位置に

## 技術メモ

- 人物検出: MediaPipe Tasks Vision の ObjectDetector + EfficientDet-Lite0(COCO)を
  `vendor/` に同梱(wasm 9.4MB + モデル 7.3MB)。**オフライン動作・外部CDN不使用**
- SIMD版wasmのみ同梱(iOS 16.4+/モダンブラウザ対応。非SIMDの古いブラウザは非対応)
- GPUデリゲートを試し、失敗時はCPUにフォールバック
- 検出は約8fpsに間引き(カウント用途には十分、発熱・電池を抑える)
- カメラ利用のため**HTTPS必須**(GitHub Pages配信ならOK)
- 記録は localStorage `crowd.v1`(上限2万点で古い順に破棄)
