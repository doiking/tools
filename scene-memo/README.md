# シーンメモ — カチンコ 🎬

撮影現場でカチンコのように使える、シーン・カット・テイクのメモアプリです。
iPad(Safari)での利用を想定していますが、スマホ・PCのブラウザでも動きます。
**ビルド不要・サーバー不要の単一HTML + PWA** 構成で、一度開けばオフラインでも使えます。

## 主な機能

- **カチン!ボタン**(または上部のシマ模様をタップ)で、現在のシーン/カット/テイクを時刻付きで記録
  - クラップ音(合成音)+ 白フラッシュで視覚・聴覚フィードバック
  - 記録後、テイク番号は自動で +1
- **シーン / カット / テイクの大きなカウンター**(±ボタン、直接入力も可。シーンは「12A」のような枝番もOK)
- **「次のカット」「次のシーン」** ボタンで カット+1・テイク1 / シーン+1・カット1・テイク1 に進む
- **直前テイクの判定**(OK / KEEP / NG)とメモ入力
- **テイク履歴**:一覧表示、判定の変更(バッジをタップで切替)、メモ編集、行削除
- **CSV書き出し / 共有**(iPadでは共有シートからAirDrop・メール等に渡せます)
- 作品名・監督・撮影・カウンター・履歴はすべて **localStorageに自動保存**
- 画面スリープ防止(Wake Lock、対応ブラウザのみ)・現在時刻表示

## 使い方(iPad)

1. このフォルダを任意の静的ホスティング(GitHub Pages など)に置き、iPadのSafariで開く
2. 共有ボタン → **「ホーム画面に追加」** でフルスクリーンのアプリとして起動できます
3. 以降はオフラインでも動作します(Service Workerがキャッシュ)

> ローカルで試す場合: `cd scene-memo && python3 -m http.server 8000` → `http://localhost:8000`
> ※ Service Worker(オフライン化)は https または localhost でのみ有効です。

## GitHub Pages で公開する場合

リポジトリの Settings → Pages → Branch を `main` / root に設定すると、
`https://<user>.github.io/tools/scene-memo/` でアクセスできます。

## ファイル構成

| ファイル | 内容 |
| --- | --- |
| `index.html` | アプリ本体(HTML/CSS/JSすべて込み) |
| `manifest.webmanifest` | PWAマニフェスト(ホーム画面追加用) |
| `sw.js` | Service Worker(オフラインキャッシュ) |
| `icons/` | アプリアイコン(スクリプト生成のカチンコ柄PNG) |

## データについて

- 記録はブラウザの localStorage にのみ保存されます(外部送信なし)
- Safariの履歴消去やサイトデータ削除で消えるため、撮影後は **CSV書き出し** での保全をおすすめします
