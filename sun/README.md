# 日の出・マジックアワー 🌅

ロケ地と日付から、日の出・日の入・ゴールデンアワー・ブルーアワーを計算するアプリです。
NOAAの太陽位置計算式による**完全オフライン計算**なので、電波のないロケ地でも使えます。

## 主な機能

- **日の出・日の入時刻と方角**(16方位+度数。カメラポジション検討用)
- **朝夕のゴールデンアワー / ブルーアワー**(太陽高度 −4°〜+6° / −6°〜−4° で定義)
- **市民薄明・南中時刻・昼の長さ**
- **24時間タイムライン**(夜→ブルー→ゴールデン→昼を色分け、今日なら現在位置マーカー付き)
- ロケ地は主要都市プリセット / GPS現在地 / 緯度経度の直接入力
- 日付は ◀▶ で前後の日に送り(ロケハン日と撮影日の比較に)
- 設定はlocalStorageに自動保存、PWA対応(ホーム画面追加・オフライン動作)

## 精度について

- NOAAの簡易級数式を使用、時刻は±1〜2分程度の精度です
- 時刻は端末のタイムゾーンで表示します(海外ロケの場合は注意)
- 方角は真北基準です(方位磁針の磁北とは西日本で約7〜8°ずれます)
