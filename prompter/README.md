# テレプロンプター 📜

撮影現場向けのテレプロンプターアプリです。iPad(Safari)想定、スマホ・PCでも動きます。
シーンメモと同じく **単一HTML + PWA** 構成で、一度開けばオフラインでも使えます。

## 主な機能

- **原稿リスト**:複数の原稿を保存・複製・削除(localStorageに自動保存)
- **自動スクロール再生**:3秒カウントダウン後に開始。画面タップで再生/一時停止
- **速さ・文字サイズを再生中に調整**(設定は記憶されます)
- **ミラー反転**:左右(⇄)・上下(⇅)。プロンプターミラー(ビームスプリッター)越しの撮影に対応
- **読み位置ガイド**(左端の▶)と**残り時間表示**・進行バー
- **Bluetoothキーボード / プレゼン用リモコン対応**:
  - スペース / Enter / PgDn / B … 再生・停止
  - ← → … 速さ調整 ・ ↑ ↓ … 手動送り ・ PgUp … 半画面戻る
  - + − … 文字サイズ ・ M … ミラー ・ Home … 先頭へ ・ Esc … 終了
- 文字数と読み上げ目安時間(300字/分)を編集画面に表示
- 画面スリープ防止(Wake Lock、対応ブラウザのみ)

## 使い方

1. 編集画面で原稿をペーストし「▶ 再生」
2. 出演者の読みやすさに合わせて「速さ」と「A+/A−」を調整
3. ホーム画面に追加すればフルスクリーンのアプリとして使えます

> ローカル確認: `cd prompter && python3 -m http.server 8000`
