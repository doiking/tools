# Vアバター 🐣

VTuber「ふう」のマスコットWebアプリ。テキストを入力すると吹き出しでしゃべり、
しゃべっている間は口パク。常時まばたき・ゆらゆら・呼吸のアイドルアニメ付き。
配信・収録の画面素材として使うことを想定。

## 使い方

1. 下の入力欄にセリフを入れて Enter(または 💬)→ 吹き出しがタイプライター表示
2. 😊 😲 💢 ✨ ボタンでエモート(にこにこ/びっくり/ぷんぷん/キラキラ)
3. 👁 で UI を隠す(クリックか Esc で復帰)。URL に `?ui=0` を付けると最初から非表示

### OBS に取り込む

- **透過(おすすめ)**: OBS の「ブラウザソース」にこのページの URL を指定。
  背景=透過のままでアバターだけが抜けて重なります(市松模様はプレビュー表示のみ)
- **クロマ単色**: 設定で グリーン/ブルー/マゼンタ/カスタム色 を選び、
  画面キャプチャ+クロマキーで抜く

### 設定(⚙️)

- キャラクター切り替え(🐣ひよこ / 🧑人間)
- 背景(透過/クロマ単色)・大きさ・横位置・吹き出し文字サイズ
- 名前(ネームプレート表示)
- セリフ読み上げ(OS内蔵の音声合成・オフライン動作)
- **キャラ画像の差し替え**: 自作PNG 3枚(ベース/口あき/目とじ)で
  オリジナルキャラにできます。3枚とも同じキャンバスサイズで作成すること

## 技術メモ

- 単一HTML・vanilla JS・外部依存なし(オフライン動作、PWA)
- 内蔵キャラ(ひよこ/人間)はSVG。表情は各SVG内のクラス付きグループの表示切替、動きは requestAnimationFrame
- 差し替え画像・設定は localStorage のみ(外部送信なし)
  - `vtuber.v1` = 設定 / `vtuber.img.v1` = 差し替え画像(dataURL)
- 読み上げは Web Speech API(speechSynthesis)。ブラウザ/OS内蔵音声を使用
