# レンズ画角シミュレーター 🔍

センサーサイズと焦点距離から、画角・撮影範囲・被写界深度を計算するアプリです。
ロケハンで「この部屋で全身入るか?」「何mm持っていくか?」を検討する用途を想定しています。
完全オフライン計算・PWA対応。

## 主な機能

- **センサープリセット**:フルサイズ / Super35 / APS-C / マイクロフォーサーズ / 1型 / 1/2.3型 / カスタム(mm直接入力)
- **16:9クロップ計算**(動画撮影モード、デフォルトON)
- **水平・垂直・対角画角**、**35mm判換算焦点距離**(同アスペクト比較)
- **撮影範囲**:被写体距離での写る範囲(横m × 縦m)
- **フレーミングプレビュー**:身長170cmの人物がどう写るかを図示
  (全身/すね上/膝上/腰上/バストアップ/顔アップを自動判定、三分割線・1mスケール付き)
- **被写界深度**:ピントが合う範囲(近点〜遠点)、過焦点距離(絞りF1.4〜F22)

## 計算について

- 薄レンズ近似です。接写域やフォーカスブリージングの大きいレンズでは実際と多少ずれます
- 許容錯乱円 = センサー対角 ÷ 1442(フルサイズで約0.030mm)
- フレーミング判定は頭上に8%のヘッドルームを取る前提です
