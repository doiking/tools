# 絵コンテ 🎞

カット表形式の絵コンテ作成ツールです。ショットデザイナーと連携できます。
完全オフライン動作・PWA対応。

## 主な機能

- **カットカード**:画像(16:9)+アクション/カメラ+セリフ/音+秒数
  - ◀▶で並べ替え、複製・削除、カット番号と合計尺は自動計算
  - 画像はタップしてアップロード(自動で960×540に縮小)。ロケハン写真などもOK
- **ショットデザイナー連携(双方向)**
  - 各カットの「🎥 ショットデザイナーでつくる」→ そのカット専用の3Dシーンが開き、
    「絵コンテのカットへ画を反映」で元のカットに画がはまって戻ってくる(リンク保持、再編集OK)
  - ショットデザイナーの☰「🎬 絵コンテへ送る」で新規カットとして追加も可能
- **印刷 / PDF出力**:🖨ボタンで絵コンテ表(No/画/アクション/セリフ/秒)を印刷。
  ブラウザの印刷ダイアログから「PDFに保存」でPDF納品もできます
- **複数ボード**の保存・複製・切替(localStorage自動保存)
- **JSON保存/読込**:バックアップや他のPC・iPadへの受け渡しに

## データ容量について

画像はブラウザのlocalStorage(約5MB)に保存されます。画像入りのカットが
数十を超える場合は、完成したボードをJSON保存してから古いボードを削除してください。
