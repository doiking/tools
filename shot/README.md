# ショットデザイナー 🎥

マネキン(人物)・カメラ・小道具を配置して、撮影カットの構図をつくるプランニングツールです。
俯瞰図でブロッキングを決め、カメラビューで実際のフレーミングを確認できます。
Three.js同梱(vendor/)・完全オフライン動作・PWA対応。

## 主な機能

- **俯瞰ビュー(左)**:1マス=1mのグリッド上に、ドラッグで自由に配置
  - 人物(色分け・ラベルA/B/C…・身長調整・立ち/座り)
  - カメラ(複数台OK。画角コーンをリアルタイム表示)
  - 箱(机や台などの小道具。幅・奥行・高さ調整)
  - 空白ドラッグで画面移動、ピンチ/±ボタンでズーム
- **カメラビュー(右)**:選択中のカメラから見た3D表示
  - 焦点距離(12〜200mm、35mm判16:9換算)・カメラ高さ・パン・チルト
  - 「近くの人物に向ける」ワンタップ・エイム
  - 三分割線オーバーレイ、複数カメラの切替(「このカメラで見る」)
- **カット管理**:複数カットの保存・複製・切替(localStorage自動保存)
- **PNG書き出し**:カメラビューをそのまま画像として保存 → 絵コンテや香盤に貼れる

## 使い方の例

1. 人物を2人置いて向かい合わせる(対談シーン)
2. カメラを置いて「近くの人物に向ける」→ 焦点距離を85mmに
3. カメラビューでバストアップになる距離までカメラをドラッグ
4. PNG書き出し → カット表に貼り付け

## 技術メモ

- 3D描画は Three.js(MITライセンス、`vendor/` に同梱)
- 画角は35mm判16:9センサー(36×20.25mm)換算。レンズ画角シミュレーターと同じ計算です
