# AEエクスプレッション集 ⚡

After Effectsの定番エクスプレッションを集めたリファレンスアプリです。
完全オフライン動作・PWA対応。

## 主な機能

- **定番30種**をカテゴリ分け(揺れ・ランダム/ループ/バウンス/追従・遅延/テキスト・数字/配置・整列/回転・軌道/時間・その他)
- **検索・カテゴリ絞り込み・★お気に入り**
- **パラメータをその場で調整**(wiggleの回数・振れ幅、バウンスの強さ等)→ 調整済みのコードをワンタップコピー
- **日本語版AE ⇄ 英語版AE 切替**:「スライダー制御」⇄「Slider Control」などエフェクト名を自動変換
- **マイ・スニペット**:自作・案件用のエクスプレッションを端末内に保存(タイトル+コード)

## 使い方

1. 探す(検索 or カテゴリ)→ 必要ならパラメータ調整 → コピー
2. AEでプロパティを Alt(Opt)+クリック → エクスプレッション欄にペースト
3. レイヤー名(「リーダー」「ターゲット」「テキスト」等)は自分の構成に合わせて書き換え
